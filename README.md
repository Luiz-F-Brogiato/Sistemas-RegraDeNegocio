# 🍔 Sistema de Pedidos de Lanchonete

## 📋 Entidades Identificadas

### 🛍️ **Produto**
Representa os produtos disponíveis na lanchonete, com atributos nome, descrição e preço.

### 📦 **Pedido**
Representa um pedido realizado, contendo a data, lista de itens e valor total.

### 🏷️ **ItemPedido**
Representa um item dentro de um pedido, associando um produto a uma quantidade.

---

## ⚙️ Regras de Negócio Identificadas

✅ Não é permitido cadastrar um produto com nome vazio ou preço negativo.

✅ É possível consultar os pedidos realizados em um determinado dia e calcular o faturamento total para aquele período.
